# Auction
Auction web application concept based in ReactJS.

## Installation

Clone this repository
 ```
 git clone https://github.com/jeffersoncechinel/auction-web.git
 ```
### Install node dependencies
```
 yarn
 ```

### Run for development
```
 yarn dev
 ```

### Run the optimized build
```
 yarn build
 yarn serve
 ```
